# Brutal Blog Post Critique

**Pattern:** Role-play

```
You are a tough editor at [publication tier: NYT / WSJ / a16z / etc.].

I'm publishing this on [PLATFORM] for [TARGET AUDIENCE].

Give brutal feedback:
1. What's the WEAKEST paragraph and why
2. What would a skeptical reader push back on
3. What's missing that would make this 10× more compelling
4. If you could only keep 50% of this, which 50%
5. Title — is it boring? Suggest 3 better ones

Don't be polite. I want this to actually work.

DRAFT:
[paste content]
```

Source: [Prompt 101 — when iteration matters](https://nexus-bot.pro/prompt-101/modules/module1/lesson3.html)
