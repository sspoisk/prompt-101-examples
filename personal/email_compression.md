# Email Compression

**Pattern:** Structured output + Few-shot

```
Compress this email to:
- 30% shorter
- Direct (no "I hope this finds you well")
- Clear ask in the first 2 sentences
- One specific next action at the end

Send 3 versions:
- Variant A: formal
- Variant B: friendly
- Variant C: ultra-short (≤50 words)

ORIGINAL:
[paste]
```

Source: [Prompt 101 RCTF Framework](https://nexus-bot.pro/prompt-101/modules/module1/lesson1.html)
