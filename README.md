# Prompt 101 Examples — Production-Ready Prompts

Practical, copy-paste prompts from the [free Prompt 101 mini-course](https://nexus-bot.pro/prompt-101/) by [NEXUS Algo Academy](https://nexus-bot.pro/).

All prompts follow the **RCTF framework** (Role + Context + Task + Format) and use one or more of the [5 core patterns](https://nexus-bot.pro/prompt-101/modules/module1/lesson2.html): zero-shot, few-shot, chain-of-thought, role-play, structured output.

## Categories

- `business/` — Sales, marketing, customer success
- `code/` — Review, debugging, refactoring
- `content/` — Writing, editing, research
- `data/` — Analysis, SQL, visualization
- `personal/` — Email, calendar, decision-making

## How to use

1. Pick a prompt
2. Replace `[BRACKETED VARS]` with your context
3. Paste into Claude/ChatGPT/Gemini
4. Iterate 2-3 times for best results
